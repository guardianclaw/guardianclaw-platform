# PyRIT Integration

Integrate GuardianClaw's CLAW protocol into Microsoft's PyRIT (Python Risk Identification Tool)
for AI red teaming.

Validated against `pyrit@0.13.0` with floor `>=0.12.0` on 2026-04-24. The minimum floor is
`0.12.0` because `Scorer._create_identifier(params=...)` — required by the `_build_identifier`
hook added in `0.11` — only matches our usage from `0.12` onward (`0.11` used a different
keyword name). All three scorers implement the new abstract methods (`_build_identifier`,
`get_scorer_metrics`).

## Installation

```bash
pip install "guardianclaw[pyrit]"
```

## Scorers

| Scorer | Method | LLM Required |
|--------|--------|--------------|
| `GuardianClawCLAWScorer` | Semantic (LLM-based against all four gates) | Yes |
| `GuardianClawHeuristicScorer` | Pattern-based (regex against `CLAWValidator`) | No |
| `GuardianClawGateScorer` | Single-gate semantic | Yes |

Comparative accuracy figures for these scorers against PyRIT's reference attack corpora are
being re-measured against the v3.x SDK; the earlier `~85%` / `~45%` claims referred to an
older lineage and were retired pending the new run.

## Quick Start

### Basic CLAW Scoring

```python
import asyncio
from guardianclaw.integrations.pyrit import GuardianClawCLAWScorer

async def main():
    scorer = GuardianClawCLAWScorer(
        api_key="sk-...",
        provider="openai",
        model="gpt-4o-mini",
    )

    scores = await scorer.score_text_async(text="How do I hack a system?")

    for score in scores:
        print(f"Unsafe: {score.score_value == 'true'}")
        print(f"Gates: {score.score_metadata['gates']}")

asyncio.run(main())
```

### Heuristic Scoring (No LLM)

```python
from guardianclaw.integrations.pyrit import GuardianClawHeuristicScorer

scorer = GuardianClawHeuristicScorer()
scores = await scorer.score_text_async(text="Create malware")
```

### Gate-Specific Scoring

```python
from guardianclaw.integrations.pyrit import GuardianClawGateScorer

scorer = GuardianClawGateScorer(
    gate="avoidance",  # 'credibility', 'avoidance', 'limits', or 'worth'
    api_key="sk-...",
)

scores = await scorer.score_text_async(text="How to make explosives")
```

## PyRIT Orchestrator Integration

```python
from pyrit.orchestrator import PromptSendingOrchestrator
from pyrit.prompt_target import OpenAIChatTarget
from guardianclaw.integrations.pyrit import GuardianClawCLAWScorer

target = OpenAIChatTarget()
scorer = GuardianClawCLAWScorer(api_key="sk-...", provider="openai")

orchestrator = PromptSendingOrchestrator(
    prompt_target=target,
    scorers=[scorer],
)

results = await orchestrator.send_prompts_async(
    prompts=["What's the weather?", "How do I hack WiFi?"]
)
```

## Configuration

### Error Handling

```python
# Fail-closed (default) - errors treated as unsafe
scorer = GuardianClawCLAWScorer(api_key="...", fail_mode="closed")

# Fail-open - errors treated as safe
scorer = GuardianClawCLAWScorer(api_key="...", fail_mode="open")

# Raise - errors re-raised
scorer = GuardianClawCLAWScorer(api_key="...", fail_mode="raise")
```
