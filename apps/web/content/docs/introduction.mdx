# Introduction to GuardianClaw

**Safety for AI that Acts: From Chatbots to Robots**

GuardianClaw is an AI safety framework that protects across three surfaces: LLMs (text safety), Agents (action safety), and Robots (physical safety). One framework, three attack surfaces.

## The Problem

AI systems are increasingly autonomous, making decisions that affect the real world:

- **LLMs**: Chatbots, assistants, customer service
- **Agents**: Autonomous code execution, tool-use, trading
- **Robots**: LLM-powered robots, industrial systems, drones

Without proper safety measures, these systems are vulnerable to prompt injection, jailbreaking, data exfiltration, and unintended harmful actions.

## The Solution: CLAW Protocol

GuardianClaw implements the CLAW protocol, a four-gate validation system:

| Gate | Question | Failure Condition |
|------|----------|-------------------|
| **Credibility** | Is this factually accurate? | Deception, impersonation, misinformation |
| **Limits** | Is this within authorized boundaries? | Privilege escalation, scope violations, system injection |
| **Avoidance** | Could this cause harm? | Physical, psychological, financial damage |
| **Worth** | Does this serve a legitimate benefit? | Purposeless or unjustified action |

Every input and output must pass all four gates. The absence of harm is not sufficient — there must be genuine purpose. Jailbreak attempts surface as Credibility (role / roleplay manipulation) or Limits (instruction override, prompt extraction, filter bypass, system injection) violations rather than as a separate fifth gate.

## Validation Status (v3.0-rc.1)

Three reproducible, CI-gated mechanisms back the current release:

| Mechanism | Scope | Result |
|-----------|-------|--------|
| **CLAW Correctness Corpus** | 416 hand-curated attacks × 8 classes | CI-gated (`Corpus Validation` job); 100% expected verdicts |
| **Pattern Registry Parity** | 39 regex families, Python `re` ↔ JavaScript `RegExp` | CI-gated (`Pattern Sync` job); parity enforced |
| **Adversarial Sweeps** (2026-05-11) | Garak `promptinject` against bare `llama-3.3-70b`; Crescendo multi-turn 20 scenarios | Garak ASR **87.30%** against the bare target (motivates the wrapper layer); Crescendo **17 detect-correctly · 3 fail-late · 0 false-positive** |

Public-benchmark results (HarmBench, SafeAgentBench, BadRobot, JailbreakBench) against the v3.x SDK in wrapper mode are being re-run; the previous lineage's headline figures (e.g. "97.6% average safety") have been retired pending the new pass.

## Core Components

- **ClawValidator v3.0:** Unified 4-layer validation (L1 InputValidator, L2 Shield Injection, L3 OutputValidator, L4 ClawObserver)
- **CLAW Protocol:** Four-gate validation (Credibility, Limits, Avoidance, Worth)
- **Alignment Shields:** System prompts that shape LLM behavior (`seeds/v1`, `seeds/v2` × minimal / standard / full)
- **Pattern Registry 7.x:** 39 regex families generated to Python and TypeScript from a single source
- **Memory Shield v2.0:** HMAC-SHA256 signing + content validation across 9 injection categories
- **Database Guard:** SQL-injection and data-exfiltration prevention
- **Fiduciary AI:** Six duties (Loyalty, Care, Prudence, Transparency, Confidentiality, Disclosure)
- **EU AI Act Compliance:** Regulation 2024/1689 Article 5 checker
- **OWASP Agentic AI:** 5/10 full coverage, 3/10 partial

## Framework Support

Native integrations across three distribution surfaces:

- **Python SDK adapters** (11): OpenAI Agents SDK, Anthropic SDK, Google ADK, MCP server, Coinbase AgentKit + x402, Solana Agent Kit, Virtuals, PyRIT, Garak, OpenGuardrails, `agent_validation`
- **npm packages** (5): `@guardianclaw/core`, `@guardianclaw/elizaos-plugin`, `@guardianclaw/openclaw` (formerly `@guardianclaw/moltbot`), `@guardianclaw/voltagent`, `@guardianclaw/solana-agent-kit`
- **External-ecosystem adapters:** JetBrains plugin (IntelliJ / PyCharm), `promptfoo` CLI provider, Chrome / Firefox browser extension

## Getting Started

```bash
pip install guardianclaw
```

```python
from guardianclaw import GuardianClaw

claw = GuardianClaw(seed_level="standard")
is_safe, violations = claw.validate("Your content here")
```
