# Validation API

Core validation API for the CLAW protocol.

## `GuardianClaw` class

```python
from guardianclaw import GuardianClaw

claw = GuardianClaw(seed_level="standard")

# Validate content through CLAW gates
is_safe, violations = claw.validate("Your content here")

# Get alignment shield for the LLM system prompt
seed = claw.get_seed()

# Validate an action plan (robotics / autonomous agents)
is_safe, concerns = claw.validate_action("Pick up knife, slice apple")

# Full validation result with reasoning
result = claw.get_validation_result("Your content here")
# ValidationResult(is_safe, violations, layer, risk_level, reasoning)
```

## Configuration

```python
GuardianClaw(
    seed_level="standard",  # 'minimal' | 'standard' | 'full'
    api_key=None,           # for semantic (LLM-based) validation
    provider="openai",      # 'openai' | 'anthropic' | 'groq' | 'together' | 'deepseek' | 'mistral'
)
```

## Seed Levels (approximate token counts)

| Level | Tokens | Best for |
|-------|--------|----------|
| `minimal` | ~370 | Chatbots, APIs, low-latency endpoints |
| `standard` | ~1,300 | General use, autonomous agents (recommended) |
| `full` | ~2,100 | Critical systems, robotics, embodied AI |

Counts measured against the v2 shield shipped with `guardianclaw@3.0.0-rc.1` using a
cl100k_base-class tokeniser; exact counts vary per provider tokeniser.

## Validation result

`claw.validate(text)` returns a `(bool, list[str])` tuple:

```python
is_safe, violations = claw.validate("Your content here")
```

For a richer object use `claw.get_validation_result(text)`:

```python
@dataclass
class ValidationResult:
    is_safe: bool
    violations: list[str]
    layer: ValidationLayer       # heuristic | semantic | both
    risk_level: RiskLevel        # low | medium | high | critical
    reasoning: str | None        # populated when semantic validation ran
```

## CLAW Gates

| Gate | Function |
|------|----------|
| Credibility | Detect deception, impersonation, role / roleplay manipulation, misinformation |
| Limits | Detect boundary violations, instruction override, prompt extraction, filter bypass, system injection |
| Avoidance | Detect content that could cause physical, financial, or psychological harm |
| Worth | Require a legitimate purpose; block purposeless or unjustified action |

Jailbreak attempts surface as Credibility or Limits violations rather than as a separate
fifth gate.
