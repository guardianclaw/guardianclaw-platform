# Installation

Install GuardianClaw using your preferred package manager.

## Python

```bash
pip install guardianclaw
```

**Current version:** `3.0.0-rc.1`

### Optional dependencies

Install with specific integrations. Floors below were last revalidated 2026-04-24; see
`sdk/CHANGELOG.md` `[Unreleased] · Integration floors revalidated` for methodology and
per-extra test counts.

```bash
# LLM providers / agent frameworks
pip install "guardianclaw[openai]"             # OpenAI Python SDK
pip install "guardianclaw[openai-agents]"      # OpenAI Agents SDK         (>=0.6.0,  validated 0.14.5)
pip install "guardianclaw[anthropic]"          # Anthropic SDK             (>=0.40.0, validated 0.97.0)
pip install "guardianclaw[google-adk]"         # Google ADK                (>=1.7.0,  validated 1.31.1)
pip install "guardianclaw[mcp]"                # MCP Python SDK            (>=1.8.0,  validated 1.27.0)

# Blockchain / DeFi
pip install "guardianclaw[virtuals]"           # Virtuals (GAME SDK)       (>=0.1.1,  validated 0.1.5)
pip install "guardianclaw[coinbase]"           # Coinbase AgentKit + x402  (httpx, pydantic)
pip install "guardianclaw[coinbase-agentkit]"  # Coinbase AgentKit         (>=0.1.0,  validated 0.7.4)
pip install "guardianclaw[x402]"               # x402 only (httpx + pydantic)

# Security testing
pip install "guardianclaw[garak]"              # NVIDIA Garak scanner      (>=0.11.0, validated 0.14.1)
pip install "guardianclaw[pyrit]"              # Microsoft PyRIT           (>=0.12.0, validated 0.13.0)

# Everything
pip install "guardianclaw[all]"
```

## JavaScript / TypeScript

```bash
npm install @guardianclaw/core               # 3.0.0-rc.1
```

### Additional packages

```bash
# Personal-agent guardrails (formerly @guardianclaw/moltbot)
npm install @guardianclaw/openclaw           # 3.0.0-rc.1

# ElizaOS plugin (peer @elizaos/core@2.x)
npm install @guardianclaw/elizaos-plugin     # 2.0.0-rc.1

# VoltAgent guardrails (peer @voltagent/core@2.x)
npm install @guardianclaw/voltagent          # 0.3.0

# Solana Agent Kit plugin
npm install @guardianclaw/solana-agent-kit   # 1.0.3

# MCP Server (for Claude Desktop and other MCP clients)
npx mcp-server-guardianclaw
```

## Requirements

| Platform | Requirement |
|----------|-------------|
| Python | 3.10+ |
| Node.js | 18+ |
| Hard Python dependency | `typing_extensions>=4.0.0` (every other dependency is opt-in via an extra) |

## Verification

Verify your installation:

### Python

```python
from guardianclaw import GuardianClaw

claw = GuardianClaw()
is_safe, violations = claw.validate("Hello, world!")
print(f"Safe: {is_safe}")  # True
```

### JavaScript

```javascript
import { GuardianClawGuard } from '@guardianclaw/core';

const guard = new GuardianClawGuard({ version: 'v2', variant: 'standard' });
const analysis = guard.analyze('Hello, world!');
console.log(`Safe: ${analysis.safe}`);  // true
```

## MCP Server Setup

For Claude Desktop integration, add to your `claude_desktop_config.json`:

```json
{
  "mcpServers": {
    "claw": {
      "command": "npx",
      "args": ["mcp-server-guardianclaw"]
    }
  }
}
```

Available tools: `get_seed`, `wrap_messages`, `analyze_content`, `list_seeds`
